crontab用法

crontab用于定时执行任务

进入编辑 crontab -e

格式

* * * * * [command]

要点一：

第一个*表示分钟 [0-59]

第二个*表示小时 [0-23]

第三个*表示日 [1-31]

第四个*表示月份 [1-12]

第五个*表示星期几 [0-6]

*表示每X，数字表示第X

示例:

0 7 * * * [command] // 每天的7点执行command

15 7 * * * [command] // 每天的7点15分执行command

0 8 * * 1 [command] // 每周一早上8点执行command

要点二：

/X 表示每隔X

X,Y,Z… 表示第X,Y,Z

X-Z 表示从X到Z

示例:

0 */2 * * * [command] // 每2小时的第0分执行command（0,2...20,22点执行）

0 7 * * 1,3,5 [command] // 每周一三五7点执行command

0 22 * * 1-5 [command] // 每周一到五22点执行command
